What gets imported

The importer uses each row's Scryfall ID to add cards by exact printing, and honors the Foil column exactly as Manabox labels each card. Ignored columns: Set name, Rarity, ManaBox ID, Purchase price, Misprint, Altered, Condition, Language, Purchase price currency, Added.

Cards not already in the local cache are fetched from Scryfall in bulk (up to 75 per request).
